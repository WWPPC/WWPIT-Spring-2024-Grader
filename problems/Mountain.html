Jie is climbing a mountain which is $N$ units tall, as inspiration for his latest game “Mountain Guarder”.

Fortunately, he has the help of an expert climbing team of robots. He can pay a $X$ dollars for a team of infinite robots with experience level $X$.

Both Jie and all of the robots start at height $0$. In $1$ second, Jie or a robot can move $1$ unit up if there is a robot below or at their height that is at most $X$ units below. Only one climber can move in each second. Note that there must be a robot within $X$ units below before the move is made, not necessarily after.

Find the minimum amount of money Jie needs to pay to climb the mountain in at most $K$ seconds.

Input

The first line contains an integer $T$ $(1 \le T \le 2\cdot 10^5)$ the number of testcases
Each testcase contains two integers $N$ and $K$ $(1 \le N \le K \le 10^9)$.

Output

Output $T$ lines, minimum amount of money Jie needs to pay for each testcase.

Sample Input 2 5 8 12 20 Sample Output 2 5

Note

It takes exactly $8$ seconds in testcase 1 with $1$ dollar.
It takes $19$ seconds in testcase 2 with $4$ dollars.

Subtask 1: $N \le 10^3$
Subtask 2: $N \le 10^6$
Subtask 3: No additional constraints
